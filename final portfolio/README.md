# Professional Personal Portfolio

A modern, fully responsive, and professional personal portfolio website built using only **HTML5, CSS3, and Vanilla JavaScript**.

## 🚀 Features
- **Modern Design**: Clean layout with professional typography and color scheme.
- **Fully Responsive**: Optimized for mobile, tablet, and desktop.
- **Interactive Elements**: Sticky navigation, hamburger menu, and scroll-to-top button.
- **Smooth Animations**: Scroll-reveal effects and smooth scrolling navigation.
- **Form Validation**: Client-side validation for the contact form.
- **SEO Optimized**: Semantic HTML and meta tags included.

## 🛠️ Local Development Instructions
You can run this project locally without any installation:

### Option 1: Simple Browser Open
1. Navigate to the project folder.
2. Double-click `index.html` to open it in your default web browser.

### Option 2: VS Code + Live Server (Recommended)
1. Open the project folder in **Visual Studio Code**.
2. Install the **Live Server** extension (by Ritwick Dey).
3. Click the **"Go Live"** button at the bottom right of the VS Code window.
4. The website will open automatically at `http://127.0.0.1:5500`.

## 🌐 Netlify Deployment Instructions
Deploying to Netlify is quick and free:

### Option 1: Drag-and-Drop (Easiest)
1. Create a free account at [Netlify](https://www.netlify.com/).
2. Go to the **"Sites"** tab.
3. Drag and drop the entire project folder into the deployment area.
4. Your site will be live in seconds!

### Option 2: GitHub Deployment (Continuous Deployment)
1. Push your project to a **GitHub** repository.
2. In Netlify, click **"Add new site"** > **"Import an existing project"**.
3. Connect your GitHub account and select your repository.
4. Click **"Deploy site"**. Netlify will automatically redeploy every time you push changes to GitHub.

### Custom Domain
- Once deployed, you can set a custom domain under **Domain Settings** in the Netlify dashboard.

## ✏️ How to Customize
I have included comments in the code to help you find where to edit:
- **Name & Title**: Search for `[Your Name]` in `index.html`.
- **Skills**: Update the `width` percentage in the `style` attribute of the `.progress` divs in `index.html`.
- **Resume**: Replace the file in `resume/resume.pdf` with your own.
- **Contact Details**: Update the `contact-info` section in `index.html`.
- **Social Links**: Update the `social-links` in the footer of `index.html`.
